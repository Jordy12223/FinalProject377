## Description
Lost Pet Finder is a web application designed to help users report and locate 
lost pets. Users can submit information about missing pets, including the animal 
type, breed, color, description, last seen location, and contact information. 
The application stores this information in a Supabase database and allows users 
to view the pet-related information through an interactive interface. The 
project also includes a Leaflet map integration for displaying pet locations 
visually, and a Swipper to see recent found pets. 

The goal of the project is to create an easy platform that helps reconnect lost 
pets with their owners through community support and shared information.
