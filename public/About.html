Helping Lost Pets Find Home

Connecting owners, shelters, and rescuers through one simple platform.

What is Lost Pet Finder?

Lost Pet Finder is a web application designed to help reunite lost pets with their owners. It centralizes lost and found pet reports into one interactive platform instead of relying on scattered posts across social media, flyers, and shelter listings.

The Problem

Right now, information about lost pets is very unorganized. Owners often post on Facebook, neighborhood apps, or contact shelters individually. This makes it difficult to track pets efficiently or search within a specific area.

Our Solution

This application uses an interactive Leaflet map to display lost and found pets in real time. Users can filter by location, distance, breed, and status to quickly find relevant information. All reports are stored in a centralized database so information stays organized and searchable.

Who This Helps

  • Pet owners looking for missing animals
  • People who have found stray pets
  • Animal shelters managing intake animals
  • Veterinary clinics assisting with microchip identification

Technology Used

  • HTML, CSS, JavaScript (frontend)
  • Leaflet.js for interactive maps
  • Dog & Cat APIs for sample animal images
  • Supabase for storing pet and user data

How It Works

  1. Users report a lost or found pet using a form
  2. The data is stored in a database (Supabase)
  3. Pets appear on an interactive map
  4. Users filter and search to locate matches
  5. Owners can reconnect with their pets faster
