# Lost Pet Finder

## Description
Lost Pet Finder is a web application designed to help users report and locate lost pets. Users can submit information about missing pets, including the animal type, breed, color, description, last seen location, and contact information. The application stores this information in a Supabase database and allows users to view pet-related information through an interactive interface. The project also includes a Leaflet map integration for displaying pet locations visually.

The goal of the project is to create an easy-to-use platform that helps reconnect lost pets with their owners through community support and shared information.

## Target Browsers
This application is mainly designed for desktop and mobile browsers, like:
- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari
The application is intended to work on:
- Windows
- macOS
- iOS
- Android

Google Chrome is the primary recommended browsers for the best compatibility and performance for this project.
